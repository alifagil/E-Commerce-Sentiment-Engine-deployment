Behind the Engine

Teknologi di Balik Layar

E-Commerce Sentiment Engine AI dibangun di atas fondasi algoritma machine learning klasik yang terbukti andal untuk klasifikasi teks — ringan, cepat, dan interpretatif.

📊

TF-IDF Vectorizer

Term Frequency–Inverse Document Frequency mengekstraksi bobot kata dari hasil preprocessing teks menjadi representasi matriks numerik yang informatif, sebagai basis vektor input matematis sebelum dieksekusi oleh mesin pengklasifikasi SVM.

Feature Extraction
🧮

Linear Support Vector Machine

Model klasifikasi berbasis hyperplane linier yang memisahkan ruang dimensi tinggi fitur secara optimal. Sangat superior untuk pengenalan pola teks e-commerce, terbukti menghasilkan performa akurasi tertinggi sebesar 94.92% pada uji eksperimen riset.

Classifier Engine
🐍

Flask REST API

Backend berbasis Flask Python mengelola pipeline inferensi data secara real-time. Alur sistem memproses Input Review melalui modul Parser Tokopedia/Shopee, dilanjutkan dengan Text Cleaning, ekstraksi fitur via TF-IDF Vectorizer, klasifikasi menggunakan Linear SVM untuk menghasilkan Sentiment Prediction dan Confidence Score yang ditransmisikan dalam wujud JSON Response menuju antarmuka web.
